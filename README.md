## Environment variables and API keys

- Keep real API keys only in local `.env` files.
- Do not commit `.env` files. This repository ignores `.env` files in all folders.
- Use `.env.example` as the template for required variables.

Quick setup:

```bash
cp .env.example .env
```

Then set your real values in `.env` locally.
